High Ubunterra

Description:


This is theme nearly looks like macOS lockscreen , with script which can set wallpaper for your desktop and simultaneously set same wallpaper on lock screen and login screen with blur effect like on macOS. In attachment you will find file ubuntu.css, where injected some changes, to make your login screen and lock screen looks more like mac OS.

ATTENTION: TESTED AND WORK ONLY ON UBUNTU 17.10 with GDM3 (xorg), 18.04.xx LTS (Any other OS based on Linux like: Lubuntu, Xbunte, Kubuntu, KDE, Manjaro, Mint and other... didn't tested!!!).

For Ubuntu 19.04 (Disco Dingo) download files:


  • High_Ubunterra_DD-2.4(noPass).tar.xz (For Ubuntu 19.04 without password (Not Safe)
  • High_Ubunterra_DD-2.4(Pass).tar.xz (For Ubuntu 19.04 with asking password)


For Ubuntu 18.10 (Cosmic Cuttlefish) download files:


  • High_Ubunterra_CC-2.3(noPass).tar.xz (For Ubuntu 18.10 without password (Not Safe)
  • High_Ubunterra_CC-2.3(Pass).tar.xz (For Ubuntu 18.10 with asking password)


For Ubuntu 17.10 and 18.04 download files:


  • High_Ubunterra_2.3(noPass).tar.xz (For Ubuntu 17.10 and 18.04 without password (Not Safe)
  • High_Ubunterra_2.3(Pass).tar.xz (For Ubuntu 17.10 and 18.04 with asking password)

  • gnome-look.org


Two versions: High_Ubunterra_X.X(noPass) - without asking password to set wallpaper but script give full permission for root folder " /usr/share/backgrounds/ " by chmod 777 (not safety). High_Ubunterra_X.X(Pass) - with asking password each time you set wallpaper for change background of " lockscreen ", it is safe, but not comfortable for user.


INSTALLATION howto:


1. Click right button of mouse and chose "Open in Terminal" in folder with install.sh, then type command ./install.sh
2. Now you just choose your image right click mouse button you will see script and "SetAsWallpaper" just click on it, wallpaper already set.
3. To uninstall use command ./uninstall.sh
Enjoy!
